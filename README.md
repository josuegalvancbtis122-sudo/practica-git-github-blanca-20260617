# Practica Git y GitHub

Proyecto individual para documentar el flujo basico de Git y GitHub:

- Crear repositorio remoto.
- Subir un proyecto con `push`.
- Actualizar con `pull`.
- Revisar diferencias entre local y remoto.
- Generar y documentar un conflicto.

## Proyecto

La aplicacion es una pagina web sencilla de presentacion para la practica.
